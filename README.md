# Modern Development — Lead Magnets

Single GitHub Pages repo hosting Modern Development lead-magnet pages.
Served at **build.moderndevelopment.co** (custom domain via CNAME).

| Path | Magnet |
|------|--------|
| `/` (index.html) | Cost-to-Build Calculator |
| `/cost-to-build.html` | Cost-to-Build Calculator (named) |

Each page is self-contained HTML with the Meta Pixel (942834204826443) and a
GoHighLevel inbound webhook wired for lead capture. Add a new magnet by dropping
a new `.html` file in the repo root.
